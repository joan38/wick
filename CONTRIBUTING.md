# Contributing

Compile:
```shell
./mill __.compile
```

Run tests:
```shell
./mill __.test
```

Publish local:
```shell
./mill __.publishLocal
```

## IDE support

### Visual Studio Code

Open this project in VS Code after downloading the [Metals plugin](https://scalameta.org/metals/docs/editors/vscode) and enjoy.

### IntelliJ IDEA

Open this project in IntelliJ IDEA after downloading the [Scala plugin](https://plugins.jetbrains.com/plugin/1347-scala) and enjoy.

### Other IDEs

Any IDE is supported as long as the IDE supports the [Build Server Protocol (BSP)](https://build-server-protocol.github.io).
